# CRM-Simplificado
Integrantes: Heitor Cabral, Isabela Tostes, Lazaro Souza

O CRM Simplificado é um sistema desenvolvido para organizar informações de clientes, oportunidades comerciais e histórico de interações, tomando como referência as necessidades da empresa 2TES. O projeto tem como objetivo oferecer uma ferramenta leve e centralizada para apoiar o acompanhamento do processo comercial. 

A solução foi construída utilizando HTML, CSS, JavaScript e o Firebase, que funciona como banco de dados principal e suporte às funcionalidades web. Paralelamente, também foi desenvolvido um modelo em MySQL, atendendo às exigências acadêmicas da disciplina de Banco de Dados. O sistema implementa operações completas de CRUD para Usuários, Clientes, Oportunidades e Interações, estruturadas com base em um modelo relacional de apoio e em coleções do Firebase utilizadas na aplicação real.
Este repositório reúne o código-fonte, scripts SQL e materiais utilizados no desenvolvimento do projeto.
